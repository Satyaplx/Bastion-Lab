## Bastion-Lab
Hands-on project demonstrating secure remote access to virtual machines using Azure Bastion in Microsoft Azure. This setup removes public IP dependency and enables safe browser-based RDP over private networking.
# Azure Bastion Project

## Overview
This project demonstrates secure access to Azure Virtual Machines using Azure Bastion without exposing public IP addresses.

## Architecture
User → Bastion → VM (Private IP)

## Key Features
- Removed public IP from VM
- Configured AzureBastionSubnet
- Deployed Azure Bastion
- Accessed VM securely via browser

## Outcome
Achieved secure remote access following Azure best practices.
